6. 이미 세팅된 컨테이너 이미지로 만들기
컨테이너 정지 : nvidia-docker stop (컨테이너 이름)
컨테이너 image파일로 변환 : nvidia-docker commit (컨테이너 이름) (저장할 이미지 이름)
image를 파일로 저장 : nvidia-docker save –o 서버경로/(저장할 파일명) (저장할 이미지 이름)
ex) nvidia-docker save –o /home/data/HK_image gpu1_190123

scp ~~

저장한 파일을 이미지로 띄우기
nvidia-docker load –i (저장한 파일명)
이후 기존의 run 명령어를 이용하여 컨테이너 띄우기
NV_GPU=0 nvidia-docker run -ti -p 8888:8888 -p 5000:5000 -p 6006:6006 -v /:/home --name=gpu0 (저장한 이미지이름)
